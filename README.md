# LightItUP!

## Welcome to **LightItUP!**
- a UP Diliman-led crowdsourced campus visibility app that helps students report lighting conditions and find safer walking paths at night. For optimal functionality, only users with a @up.edu.ph email can contribute reports, while everyone can view real-time updates.
